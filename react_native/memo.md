# react-nativeとは
[公式ドキュメント読め！！](https://reactnative.dev/docs/environment-setup)

- どうやらコミュニティを信じたほうが良さそう
- `expo`てのを信じたほうが良さそう
- `expo`はもともとreact-nativeのライブラリだったけど、今はreact-nativeのフレームワーク兼プラットフォームになったらしい。つまり、**今と昔で`expo`の意味が違う**
