im920関連についてここに示す。

# im920sl
他のim920シリーズと互換性がない（多分）

## コマンド
- `TXDU`: ユニキャスト送信
```:送信
TXDU [相手ノード番号hex] [データhex]\n\r
```
```:受信
00,[送信元ノード番号hex],[よくわからん]:[データhex（1byteごとに,カンマ）]\n\r
```

# 交流ロボコン2025設定
im920slを使用する
- 親機モジュール
  - シリアル番号: `00112434`
  - ノード番号: `0x0001`
  - 設定コマンド
  ```
  ENWR
  OK
  STNN 0001
  RPRM
  SBRT 7
  ```
- 子機モジュール
  - シリアル番号: `00112436`
  - ノード番号: `0x2560`
  - 設定コマンド
  ```
  ENWR
  OK
  STNN 2560
  RPRM
  SBRT 7
  ```
